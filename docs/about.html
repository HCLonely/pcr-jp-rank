关于本站

  • 此站点为公主连结Re:Dive 日服节奏榜(排行榜)，榜单数据来自GameWith, 每日早晚6点从该网站抓取一次数据.
  • 角色别名数据来自HoshinoBot项目，最新的角色只会显示日文名.
  • 角色详情会跳转至公主连结 Box的详情页，有更好的角色数据来源可以在Github提交 Issue 或 PR.
  • 娱乐排行相关角色信息来自干炸里脊资源站.
  • 每月底会对当月28日的排行数据进行归档.
